## 🚀 How to run

Make sure you have the [JDK](https://www.oracle.com/java/technologies/downloads/) installed, then:

```bash
# Compile
javac basics/Hello.java

# Run
java Hello
```

> **Tip:** Run commands from inside the folder that contains the `.java` file, or adjust the path accordingly.
